# iPhone Preview App

The repository includes a source-built iPhone app target at `ios/BuretteMobile`.

The iPhone app is not part of the Homebrew macOS release. Build it from source when you want iOS document handoff through Files or another iOS document provider.

## Focus

- Apple-style file browser and document opening for molecular project files.
- Mol* structure previews in a full-screen mobile viewer.
- RDKit-rendered SDF grid and table previews.
- Trajectory and pose controls near the bottom of the viewport when a loaded document needs them.
- App icon and document-type registration for supported molecular formats.

## Source reference

See the [`ios/BuretteMobile`](https://github.com/SergeiNikolenko/Burette/tree/main/ios/BuretteMobile) source tree for target layout, signing notes, build commands, and real-device install flow.
